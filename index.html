Definiciones de CSS

1. Definición de CSS

CSS (Cascading Style Sheets) es un lenguaje utilizado para describir la presentación de un documento HTML. Define cómo los elementos deben ser visualizados en una página web, permitiendo la personalización de aspectos como colores, fuentes, márgenes, y la disposición en general.

2. Funcionamiento Básico de CSS

CSS funciona aplicando estilos a los elementos de un documento HTML a través de "reglas". Estas reglas definen cómo los elementos deben aparecer visualmente, afectando propiedades como el tamaño de la fuente, los colores, el espaciado, y más. Las reglas CSS pueden aplicarse directamente en el HTML, mediante un archivo CSS externo o en líneas dentro del mismo documento HTML.

3. Incluir CSS en un archivo HTML

CSS se puede incluir en HTML de tres formas:

  • CSS externo: Enlazando un archivo CSS separado usando la etiqueta <link> en la sección <head> del HTML.
  • CSS interno: Definiendo estilos dentro de una etiqueta <style> dentro de la sección <head>.
  • CSS en línea: Aplicando estilos directamente en los elementos HTML con el atributo style.

4. Definición de una regla CSS

Una regla CSS es una instrucción que define cómo se debe estilizar uno o varios elementos de una página web. Una regla tiene dos componentes principales: un selector que indica a qué elementos se aplicará, y un bloque de declaración que contiene una o más declaraciones de estilo.

5. Elementos de una regla CSS

  • Regla: Es la combinación de un selector y un bloque de declaración, que define qué elementos de HTML serán estilizados y cómo.
  • Selector: Indica el elemento HTML al que se aplicará la regla.
  • Bloque de declaración: Conjunto de una o más declaraciones que definen el estilo. Se encierra entre llaves { }.
  • Propiedad: El atributo del elemento HTML que se quiere estilizar.
  • Valor: Especifica el estilo que se aplicará a la propiedad.

6. Medios CSS

Los media queries permiten aplicar estilos CSS específicos dependiendo de las características del dispositivo, como el ancho de la pantalla. Esto es útil para crear sitios web adaptables o "responsivos".

7. Comentarios CSS

Los comentarios en CSS se utilizan para explicar partes del código y no son procesados por el navegador. Se escriben entre /* y */.

8. Sintaxis de la definición de las propiedades CSS

La sintaxis básica de una propiedad CSS es:

propiedad: valor;

Cada propiedad se separa del valor mediante dos puntos : y la declaración termina con un punto y coma ;.
